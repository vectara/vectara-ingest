# Contributing to Vectara Ingest

Thank you for your interest in contributing to vectara-ingest! We welcome contributions from the community.

## Ways to Contribute

- **Report Bugs**: Open an [issue](https://github.com/vectara/vectara-ingest/issues)
- **Suggest Features**: Share your ideas via issues
- **Improve Documentation**: Help make our docs better
- **Write Code**: Submit pull requests for bug fixes or new features
- **Add Crawlers**: Create new crawler implementations

## Getting Started

1. Fork the repository
2. Clone your fork
3. Create a feature branch
4. Make your changes
5. Submit a pull request

## Development Setup

```bash
git clone https://github.com/YOUR_USERNAME/vectara-ingest.git
cd vectara-ingest
pip install -r requirements.txt
```

## Code Style

- Follow PEP 8 for Python code
- Add docstrings to functions and classes
- Include type hints where appropriate

## Testing

```bash
# Run tests
pytest tests/

# Run specific test
pytest tests/test_indexer.py
```

## Creating a New Crawler

See the [Custom Crawler Guide](advanced/custom-crawler.md) for detailed instructions.

## Submitting Pull Requests

1. Ensure tests pass
2. Update documentation if needed
3. Write clear commit messages
4. Reference related issues

## Code of Conduct

Be respectful and constructive in all interactions.
